# 📸 프로젝트 소개 (Korean)
GoHard AI Upscaler는 저해상도 이미지에 새로운 생명력을 불어넣는 지능형 이미지 개선 도구입니다. 최신 AI 모델을 활용하여 디테일과 질감을 유지하면서도 선명한 고해상도 이미지를 생성합니다.

## 주요 기능
고정밀 업스케일링: 블러 현상 없이 해상도를 높이는 딥러닝 모델 탑재.

직관적인 UI: 복잡한 설정 없이 누구나 쉽게 사용 가능.

로컬 프로세싱: 모든 작업은 사용자의 PC에서 이루어지며 데이터 유출 걱정이 없습니다.

표준화된 환경: uv와 pyproject.toml을 통한 세련된 의존성 관리.

## 설치 및 실행
```Bash
# 저장소 복제
git clone https://github.com/gohard-lab/gohard-ai-upscaler.git
cd gohard-ai-upscaler

# uv를 이용한 패키지 설치
uv sync

# 실행
uv run streamlit run src/app.py
```
