# 📸 프로젝트 소개 (Korean)
GoHard AI Upscaler는 저해상도 이미지에 새로운 생명력을 불어넣는 지능형 이미지 개선 도구입니다. 최신 AI 모델을 활용하여 디테일과 질감을 유지하면서도 선명한 고해상도 이미지를 생성합니다.

## 주요 기능
고정밀 업스케일링: 블러 현상 없이 해상도를 높이는 딥러닝 모델 탑재.

직관적인 UI: 복잡한 설정 없이 누구나 쉽게 사용 가능.

로컬 프로세싱: 모든 작업은 사용자의 PC에서 이루어지며 데이터 유출 걱정이 없습니다.

표준화된 환경: uv와 pyproject.toml을 통한 세련된 의존성 관리.

## 설치 및 실행
```Bash
# 저장소 복제
git clone https://github.com/gohard-lab/gohard_ai_upscaler.git
cd gohard_ai_upscaler

# uv를 이용한 패키지 설치
uv sync

# 실행
uv run streamlit run src/app.py
```

## 구글 코랩 사용법 (사본 저장 포함)
### 0. 내 드라이브에 사본 저장 (가장 먼저 할 일)
공유된 노트북은 '읽기 전용'일 수 있습니다. 본인이 자유롭게 실행하고 수정하려면 반드시 사본을 만들어야 합니다.

상단 메뉴에서 **[파일(File)]**을 클릭합니다.
**[드라이브에 사본 저장(Save a copy in Drive)]**을 선택합니다.
새 탭으로 열린 '~의 사본' 노트북에서 작업을 진행하세요.

### 1. GPU 가속기 설정
상단 메뉴에서 런타임 > 런타임 유형 변경을 클릭합니다.
하드웨어 가속기를 T4 GPU로 선택하고 저장을 누릅니다.

### 2. 코드 실행
각 코드 블록 왼쪽에 있는 [▶] 실행 버튼을 순서대로 누릅니다.
경고창이 뜨면 **'무시하고 계속 실행'**을 선택하세요.
